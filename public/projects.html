Rotten Potatoes
Project Links
  • Rotten Potatoes
Description

Rotten Potatoes was one of my early projects during the initial Covid-19 lockdowns in Spring 2020. With not a lot to do, during this period my family and I watched a movie every night for almost two months. I created a review and ratings site so we could determine our family favourite of the quarantine. I loved this project because it was a lot of fun to work on and I also learned a lot of new skills from it such as more advanced CSS, databases, and JQuery. The current iteration was rewritten with React a few months later as a practice exercise.

Tech Stack

JavaScript, React, MongoDB, JQuery, Express
